官网：https://pear.php.net
1 pear
	PEAR是PHP扩展与应用库(the PHP Extension and Application Repository)的缩写.它是一个PHP扩展及应用的一个代码仓库.
2 download&install
	获取：https://pear.php.net/go-pear.phar（另存为go-pear.phar）
	$ php go-pear.phar
	修改php.ini,在 include_path = ".;c:\php\pear\"中加入pear目录
3 use
	pear -h  查看命令
	pear install packagename(包名)  自W路安b某一pear程式
	pear download packagename	下d packages 但不安b
	pear download-all
	pear install filename.tgz 	安b已下d的package(./go-pear-bundle/)
	pear remote-list  目前pearW站上所有可取得pear程式炝斜
	pear list		列出已安bpackage
	pear list-upgrades 列出可以升的package
	pear upgrade packagename 更新package
	pear upgrade-all 
	pear uninstall packagename  h除已安b的package
